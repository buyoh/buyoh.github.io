手抜きなヘルプ

実装メモです。

目的・きっかけ

mspaintで図を描いて座標取るのがたいへん！！

書式

  • command prm1,prm2,prm3,...,prmN
  • 各コマンドは改行区切りまたは:で区切る。
  • コマンドとパラメータの間は必ずスペースを入れる。他は任意。
  • basicとかhspっぽい

  • パラメータに与えることができる値の制約などは今後の課題（パラメータの中にスペースが入ったりすると問題)
  • 変数、式は未実装。'24 + 37'と書いても解釈されない。

命令一覧

  • pos x,y
  • color code
  • color r,g,b
  • color r,g,b,a
  • line x,y
  • line x1,y1,x2,y2
  • pset x,y
  • circle x,y,r
  • polygon n,x1,y1,x2,y2,...,xn,yn
  • scale left,top,right,bottom

未実装

  • box x,y,w,h
  • width w
